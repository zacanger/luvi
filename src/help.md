the luvi server

usage:

  luvi                       # launch default server
  luvi foo bar               # start luvi servers `foo` & `bar`
  luvi -p 1337               # port to listen on for requests
  luvi -r /path/to/www/root  # root dir to serve
  luvi -v                    # luvi version
  luvi -h                    # this help

see the readme for config options and api usage
